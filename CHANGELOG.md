# Changelog

All notable changes to Ursly VFS will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Settings tab with theme customization
- Real-time system metrics monitoring
- Onboarding tour for new users
- Auto-update functionality
- Multi-platform CI/CD builds (macOS, Windows, Linux)

### Changed

- Enhanced Spotlight Search with smart operators
- Improved keyboard shortcuts documentation
- Updated website and README with latest features

## [1.1.4] - 2024-12-25

### Added

- Multi-cloud storage support (15+ providers)
- Spotlight Search (`Cmd+K` / `Ctrl+K`)
- Video transcription with Whisper (99 languages)
- Smart tagging and organization
- Real-time system performance monitoring
- Keyboard-first navigation
- Three deployment modes (Cloud GPU, Workstation, Browser-Only)

### Changed

- Improved file browser UI/UX
- Enhanced storage integration interface
- Better error handling and user feedback

### Fixed

- Theme toggle flicker issue
- Storage connection stability
- File operation reliability

## [1.0.0] - 2024-12-01

### Added

- Initial release
- Basic file browser functionality
- Local storage support
- Dark theme
