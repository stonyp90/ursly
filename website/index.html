Installation Help

macOS: "App is damaged" error

Quick Fix: Double-click Install Ursly VFS.command in the DMG (recommended)

Manual Fix: Run in Terminal:

xattr -cr /Applications/Ursly\ VFS.app

macOS blocks apps from unidentified developers. This removes the quarantine flag.

Windows: SmartScreen warning

Quick Fix: Click "More info" then "Run anyway"

PowerShell Fix: Run as Administrator:

Unblock-File -Path "$env:USERPROFILE\Downloads\ursly-vfs.msi"

Windows SmartScreen warns about apps from unknown publishers.

Linux: Make AppImage executable

Quick Fix: Right-click AppImage, Properties, mark as executable

Terminal Fix:

chmod +x ~/Downloads/ursly-vfs*.AppImage && ~/Downloads/ursly-vfs*.AppImage

Linux requires explicit permission to run downloaded executables.
