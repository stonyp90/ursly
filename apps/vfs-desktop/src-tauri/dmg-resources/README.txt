================================================================================
                         URSLY VFS - Installation Guide
================================================================================

QUICK INSTALL:
--------------
Double-click "Install Ursly VFS.command" to automatically install the app.

MANUAL INSTALL:
---------------
1. Drag "Ursly VFS.app" to your Applications folder
2. Open Terminal and run:
   xattr -cr /Applications/Ursly\ VFS.app
3. Open Ursly VFS from Applications

WHY THE EXTRA STEP?
-------------------
macOS Gatekeeper blocks apps from unidentified developers. The xattr command
removes the "quarantine" flag that triggers this warning. This is standard
practice for apps distributed outside the Mac App Store.

Your security is not compromised - the app is safe to use.
